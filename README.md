# body_perception

1. Clone this repo
```
$ git clone https://github.com/Sri-vatsa/body_perception.git
```

2. Install dependencies
```
$ npm install
```

3. Run local playground server which will open a browser window
```
$ npm run playground
```

OR

Run dev script for backend calls
```
$ npm run dev
```
